## 🛠 **Development Setup**

### **Prerequisites**
- Node.js 18.0.0 or higher
- npm 8.0.0 or higher

### **Installation**
```bash
# Clone the repository
git clone https://github.com/bible-aura/blog.bibleaura.xyz.git
cd blog.bibleaura.xyz

# Install dependencies
npm install

# Start development server
npm run dev
```

### **Available Scripts**
```bash
npm run dev        # Start development server with Turbopack
npm run build      # Build for production
npm run start      # Start production server
npm run lint       # Run ESLint
npm run sitemap    # Generate sitemap manually
```

## 🚀 **Deployment**

### **Recommended: Vercel**
```bash
# Install Vercel CLI
npm install -g vercel

# Deploy to production
vercel --prod
```

### **Manual Deployment**
```bash
# Build the application
npm run build

# Start production server
npm start
```

### **Environment Setup**
1. **Domain Configuration**: Point `blog.bibleaura.xyz` to deployment
2. **SSL Certificate**: Configure HTTPS
3. **CDN Setup**: Enable for global performance

### **Sitemap Management**
The sitemap is automatically generated at `/sitemap.xml` with:
- All 20 blog posts with custom priorities
- Proper change frequencies based on content type
- Last modification dates

### **Robots.txt Configuration**
- Allows all search engines
- Blocks AI scrapers (GPTBot, ChatGPT-User)
- Excludes admin and API routes
